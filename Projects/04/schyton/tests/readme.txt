This directory contains test cases that are listed our schyton readme.txt

###################################################################

RUN ALL THE TESTS:

- Inside "tests/" open stk
- Enter the stk interpreter and run all the tests by loading "all.scm":

---------------------------------------

$ stk -l ../../../../lib/simply.scm

---------------------------------------

$ STK > (load "all.scm")

---------------------------------------

###################################################################

RUN ONE TEST

- Inside "tests/" open stk
- Enter the stk interpreter and run a concrete test. For example to run the test for the first exercise we load "q1.scm":

---------------------------------------

$ stk -l ../../../../lib/simply.scm

---------------------------------------

$ STK > (load "q1.scm")

---------------------------------------
